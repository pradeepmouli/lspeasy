# Functions

## `create`
Creates a new DocumentLink literal.
```ts
create(range: Range, target?: string, data?: any): DocumentLink
```
**Parameters:**
- `range: Range` — 
- `target: string` (optional) — 
- `data: any` (optional) — 
**Returns:** `DocumentLink`

## `is`
Checks whether the given literal conforms to the DocumentLink interface.
```ts
is(value: any): value is DocumentLink
```
**Parameters:**
- `value: any` — 
**Returns:** `value is DocumentLink`
