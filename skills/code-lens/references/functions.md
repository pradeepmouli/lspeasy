# Functions

## `create`
Creates a new CodeLens literal.
```ts
create(range: Range, data?: any): CodeLens
```
**Parameters:**
- `range: Range` — 
- `data: any` (optional) — 
**Returns:** `CodeLens`

## `is`
Checks whether the given literal conforms to the CodeLens interface.
```ts
is(value: any): value is CodeLens
```
**Parameters:**
- `value: any` — 
**Returns:** `value is CodeLens`
