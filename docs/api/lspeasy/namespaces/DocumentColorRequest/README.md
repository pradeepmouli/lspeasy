# DocumentColorRequest

A request to list all color symbols found in a given text document. The request's
parameter is of type [DocumentColorParams](../../../interfaces/DocumentColorParams.md) the
response is of type [ColorInformation\[\]](../../../interfaces/ColorInformation.md) or a Thenable
that resolves to such.
