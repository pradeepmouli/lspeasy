# TypeDefinitionRequest

A request to resolve the type definition locations of a symbol at a given text
document position. The request's parameter is of type [TextDocumentPositionParams](../../../interfaces/TextDocumentPositionParams.md)
the response is of type [Definition](../../../type-aliases/Definition.md) or a Thenable that resolves to such.
