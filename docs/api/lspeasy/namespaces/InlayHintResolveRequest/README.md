# InlayHintResolveRequest

A request to resolve additional properties for an inlay hint.
The request's parameter is of type [InlayHint](../../../type-aliases/InlayHint.md), the response is
of type [InlayHint](../../../type-aliases/InlayHint.md) or a Thenable that resolves to such.
