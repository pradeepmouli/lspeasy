# DefinitionRequest

A request to resolve the definition location of a symbol at a given text
document position. The request's parameter is of type TextDocumentPosition
the response is of either type [Definition](../../../type-aliases/Definition.md) or a typed array of
[DefinitionLink](../../../type-aliases/DefinitionLink.md) or a Thenable that resolves to such.
