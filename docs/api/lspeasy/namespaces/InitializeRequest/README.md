# InitializeRequest

The initialize request is sent from the client to the server.
It is sent once as the request after starting up the server.
The requests parameter is of type [InitializeParams](../../../type-aliases/InitializeParams.md)
the response if of type [InitializeResult](../../../interfaces/InitializeResult.md) of a Thenable that
resolves to such.
