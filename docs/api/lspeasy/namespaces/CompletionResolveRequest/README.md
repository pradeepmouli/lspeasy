# CompletionResolveRequest

Request to resolve additional information for a given completion item.The request's
parameter is of type [CompletionItem](../../../interfaces/CompletionItem.md) the response
is of type [CompletionItem](../../../interfaces/CompletionItem.md) or a Thenable that resolves to such.
