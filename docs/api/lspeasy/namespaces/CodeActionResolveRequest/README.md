# CodeActionResolveRequest

Request to resolve additional information for a given code action.The request's
parameter is of type [CodeAction](../../../interfaces/CodeAction.md) the response
is of type [CodeAction](../../../interfaces/CodeAction.md) or a Thenable that resolves to such.
