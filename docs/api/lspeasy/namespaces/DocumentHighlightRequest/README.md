# DocumentHighlightRequest

Request to resolve a [DocumentHighlight](../../../interfaces/DocumentHighlight.md) for a given
text document position. The request's parameter is of type TextDocumentPosition
the request response is an array of type [DocumentHighlight](../../../interfaces/DocumentHighlight.md)
or a Thenable that resolves to such.
