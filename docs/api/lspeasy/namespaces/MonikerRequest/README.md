# MonikerRequest

A request to get the moniker of a symbol at a given text document position.
The request parameter is of type [TextDocumentPositionParams](../../../interfaces/TextDocumentPositionParams.md).
The response is of type [Moniker\[\]](../../../interfaces/Moniker.md) or `null`.
