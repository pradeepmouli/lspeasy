# DocumentSymbolRequest

A request to list all symbols found in a given text document. The request's
parameter is of type [TextDocumentIdentifier](../../../interfaces/TextDocumentIdentifier.md) the
response is of type [SymbolInformation\[\]](../../../interfaces/SymbolInformation.md) or a Thenable
that resolves to such.
