# DocumentLinkResolveRequest

Request to resolve additional information for a given document link. The request's
parameter is of type [DocumentLink](../../../interfaces/DocumentLink.md) the response
is of type [DocumentLink](../../../interfaces/DocumentLink.md) or a Thenable that resolves to such.
