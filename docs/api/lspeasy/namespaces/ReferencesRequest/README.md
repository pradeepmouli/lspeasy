# ReferencesRequest

A request to resolve project-wide references for the symbol denoted
by the given text document position. The request's parameter is of
type [ReferenceParams](../../../interfaces/ReferenceParams.md) the response is of type
[Location\[\]](../../../interfaces/Location.md) or a Thenable that resolves to such.
