# InlineValueRequest

A request to provide inline values in a document. The request's parameter is of
type [InlineValueParams](../../../type-aliases/InlineValueParams.md), the response is of type
[InlineValue\[\]](../../../type-aliases/InlineValue.md) or a Thenable that resolves to such.
