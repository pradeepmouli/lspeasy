# ColorPresentationRequest

A request to list all presentation for a color. The request's
parameter is of type [ColorPresentationParams](../../../interfaces/ColorPresentationParams.md) the
response is of type [ColorInformation\[\]](../../../interfaces/ColorInformation.md) or a Thenable
that resolves to such.
